
# Project: dynip-query

Project dynip-query perl script for updating ufw on Linux.

This program works in concert with dynip-server, dynip-client.

1. Pre-requisites:

    Software:
    
    ```text
    Python 3.10.6.    
    Linux (Ubuntu 22.04.2 LTS (Jammy Jellyfish)).
    ```
2. Running

    This script runs as root.
